# DeathHistoryNotes

Minecraft mod-extension for [Notes](https://modrinth.com/mod/notes)

## Description

DeathHistoryNotes is a Minecraft mod-extension designed to enhance your gameplay by providing detailed notes about player deaths. This mod works as an add-on for the [Notes mod](https://modrinth.com/mod/notes), allowing you to keep track of all death events in your Minecraft world.

## Features

- Automatically records player death events
- Stores information such as location, time

## Requirements

- Minecraft 1.21.5-1.21.8, 1.21.10
- [Notes](https://modrinth.com/mod/notes)
- [Fabric API](https://modrinth.com/mod/fabric-api)
- [Fabric Loader](https://github.com/FabricMC/fabric-loader)

## Installation

1. Download and install the [Notes mod](https://modrinth.com/mod/notes) if you haven't already.
2. Download the latest release of DeathHistoryNotes for your Minecraft version from the [releases page](https://github.com/thVAZIK/DeathHistoryNotes/releases).
3. Place the downloaded .jar file into your Minecraft mods folder.
4. Launch Minecraft.

## Usage

- Play as usual; the mod will automatically record death events.
- Open the Notes interface to view detailed death history.
